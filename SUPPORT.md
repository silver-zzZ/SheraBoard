# 支持

请通过 GitHub Issues 反馈 bug 或提出功能建议。

提交 issue 前请先确认：

- 如果只是想运行软件，请下载 Releases 里的 exe 或 zip，不要下载源码包。
- 你的系统是 Windows 10/11。
- 如果下载的是 framework-dependent 包，电脑已安装 .NET 8 Windows Desktop Runtime。
- SheraBoard 没有已经在托盘里运行。

反馈 bug 时请说明：

- SheraBoard 版本。
- Windows 版本。
- 下载的是哪个发布包。
- 复现步骤。
- 你预期会发生什么。
- 实际发生了什么。

不要在公开 issue 里粘贴敏感剪贴板内容。
