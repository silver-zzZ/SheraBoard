# Support

Use GitHub Issues for bug reports and feature requests.

Before opening an issue, check:

- You downloaded a Release asset, not the source code zip, if you only want to run the app.
- You are running Windows 10/11.
- If using the framework-dependent package, .NET 8 Windows Desktop Runtime is installed.
- SheraBoard is not already running in the tray.

For bug reports, include:

- SheraBoard version.
- Windows version.
- Which package you downloaded.
- Reproduction steps.
- What you expected.
- What happened.

Do not paste sensitive clipboard contents into public issues.
