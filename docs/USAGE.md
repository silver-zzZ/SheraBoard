# SheraBoard 使用说明

## 安装与启动

从 GitHub Releases 下载发布包：

- `standalone` 包适合普通用户，解压后直接运行。
- `framework-dependent` 包体积更小，但需要先安装 .NET 8 Windows Desktop Runtime。

解压后运行：

```text
SheraBoard.exe
```

SheraBoard 会常驻托盘。可以从托盘菜单打开主窗口、设置窗口或退出应用。

开机自启会使用你当前实际运行的 `SheraBoard.exe` 路径。你可以把发布包解压到桌面、下载目录、D 盘或其他目录；只要从那个位置运行一次并保持“随 Windows 启动”开启，SheraBoard 会把自启项更新到当前路径。

## 常用操作

- 打开 SheraBoard 主窗口：`Win+V`
- 打开主历史窗口：点击托盘菜单中的主窗口入口
- 恢复剪贴板：在历史列表中选择记录并复制/恢复
- 收藏或置顶：在历史列表中对重要记录标记，方便之后查找
- 暂停记录：在设置或托盘菜单中暂停捕获

SheraBoard 运行时会接管 `Win+V`，因此 Windows 原生剪贴板历史面板不会响应这个快捷键。设置页里的备用快捷键默认是 `Ctrl+Alt+V`，可作为额外打开方式。

## 搜索与筛选

SheraBoard 会保存用于展示和检索的预览文本、来源应用、类型、捕获时间等信息。你可以按关键词查找，也可以结合来源应用、日期、内容类型等条件缩小范围。

## 设置

设置页包含：

- `Win+V` 主快捷键和备用快捷键。
- 是否随 Windows 启动。
- 关闭窗口后是否自动复制。
- 最大本地存储空间。
- 忽略指定进程。
- 数据存储位置。

默认数据目录：

```text
%LOCALAPPDATA%\SheraBoard
```

如果更改存储位置，SheraBoard 会把后续数据写入新的目录。

## 数据清理

SheraBoard 会根据设置中的最大存储空间清理较旧记录。收藏和置顶记录会优先保留，但仍建议不要把它当作长期备份系统使用。

需要彻底删除本地历史时，请先退出 SheraBoard，再删除数据目录。
